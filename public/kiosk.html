Set up this stick PC to auto-launch Pool Cue on boot

1 Open Command Prompt as Admin
Tap the Windows search bar, type cmd, then tap "Run as administrator"
2 Paste this command
Copy the command below, paste it into the black command window, and press Enter:
powershell -Command "$s=(New-Object -ComObject WScript.Shell).CreateShortcut(\"$env:APPDATA\Microsoft\Windows\Start Menu\Programs\Startup\PoolCue.lnk\");$s.TargetPath='C:\Program Files\Google\Chrome\Application\chrome.exe';$s.Arguments='--kiosk --noerrdialogs --disable-translate --no-first-run --disable-infobars https://pool-cue.onrender.com/board/table1';$s.Save();powercfg /change standby-timeout-ac 0;powercfg /change monitor-timeout-ac 0;echo DONE"
3 Reboot
Restart the stick PC. It will boot straight into Pool Cue fullscreen. To exit: tap the top-left corner 5 times quickly.
